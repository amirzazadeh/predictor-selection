# Predictor Selection Strategies in Regression Models

Practical, applied guide to selecting predictors in regression models using **conceptual/theory-driven** and **data-driven** strategies, with interpretation tips for **Stata** and **R** outputs.

---

## Learning objectives

By the end of this session, you will be able to:

- Explain the basics of regression models
- Understand conceptual and theory-driven approaches to predictor selection
- Choose data-driven approaches (e.g., stepwise)
- Handle multicollinearity and correlated predictors
- Understand Stata and R outputs using applied examples

## Overview

This repository demonstrates practical predictor selection workflows for regression modeling. It emphasizes:

- **Research-question-first** modeling (theory, subject-matter knowledge; DAG-informed selection when applicable)
- **Data-driven** selection approaches (e.g., stepwise) and when they can mislead
- **Collinearity-aware** modeling with correlated predictors
- Clear **interpretation of Stata and R output** using applied examples
